# Тестовое задание для Прайм 

Модуль для реализации некоторых REST-контроллеров интернет-магазина, использующий новую модель роутинга.

Задание 1
Спроектировать REST API для реализации функционала стандартного функционала «1С-Битрикс: Управление сайтом» (Бизнес).

Задание 2.
Настроить роутинг для спроектированного API.

Задание 3
Разработать модуль, реализующий контроллеры по спроектированному API

Задание 4
Разработать консольный скрипт, реализующий выгрузку товаров в формате Excel на заданный e-mail
