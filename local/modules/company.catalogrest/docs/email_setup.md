# Настройка почтового шаблона для выгрузки товаров

## Шаг 1: Создание почтового события

1. Перейдите в `Настройки` → `Настройки продукта` → `Почтовые события`
2. Нажмите "Добавить почтовое событие"
3. Заполните поля:
   - **Тип события**: CATALOG_EXPORT_TO_EXCEL
   - **Название**: Выгрузка товаров в Excel
   - **Описание**: Отправка файла с выгрузкой товаров

## Шаг 2: Создание почтового шаблона

1. В созданном событии нажмите "Добавить шаблон"
2. Заполните поля:
   - **Тема письма**: Выгрузка товаров из каталога
   - **Тип сообщения**: HTML
   - **Кодировка**: UTF-8
   - **Тело сообщения**:

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Выгрузка товаров</title>
</head>
<body>
    <p>Здравствуйте!</p>
    <p>Во вложении находится файл с выгрузкой товаров из каталога одежды.</p>
    
    <p><strong>Информация о выгрузке:</strong></p>
    <ul>
        <li>Сайт: #SITE_NAME#</li>
        <li>Дата выгрузки: #DATE#</li>
    </ul>
    
    <p>Файл содержит следующие данные:</p>
    <ol>
        <li>ID товара</li>
        <li>Наименование товара</li>
        <li>Категория товара</li>
        <li>Ссылка на товар</li>
        <li>Количество торговых предложений</li>
        <li>Минимальная цена</li>
    </ol>
    
    <p>С уважением,<br>
    Администрация сайта #SITE_NAME#</p>
</body>
</html>
